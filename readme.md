# - Cafetería

## Descripción

Este proyecto consiste en el desarrollo de una página web para una cafetería. El objetivo es ofrecer una experiencia atractiva para los clientes, permitiéndoles conocer el menú, la información del negocio y los servicios disponibles mediante una interfaz moderna y responsiva.

## Características

- Diseño responsivo.
- Página de inicio.
- Menú de bebidas y alimentos.
- Sección "Nosotros".
- Información de contacto.
- Integración con redes sociales.
- Interfaz moderna utilizando Tailwind CSS.

## Tecnologías utilizadas

- HTML5
- JavaScript (ES6)
- Node.js
- Tailwind CSS
- npm

## Requisitos

- Node.js v18 o superior
- npm

## Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/JaramospUT/proyecto_colaborativo.git
```

2. Entrar a la carpeta del proyecto:

```bash
cd PROYECTO_COLABORATIVO

3. Instalar dependencias:

```bash
npm install
```

## Ejecutar el proyecto

Si utilizas el servidor de desarrollo:

```bash
npm serve dev
```
